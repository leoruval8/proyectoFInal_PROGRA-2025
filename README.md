Este es mi proyecto final de Introducion a la Programación.

Hice un programa que simula el juego del "Ahorcado", el objetivo principal es adivinar una palabra que el programa te dá aleatoriamente antes de quedarte sin vidas, 
intentando con letras hasta que adivines la palabra completa.

Le agregue unas funciones para que puedas guardar tu partida y cargar tu partida guardada para poder volver a intentar adivinar la palabra en otro momento.

Este proyecto lo hice todo yo solo :(

Durante el desarrollo de este proyecto aprendí mas o menos como funcionan las lambdas, que pueden ser herramnientas muy utiles para diferentes casos, 
también aprendí como se estructura un codigo funcional, y sobre todo lo mas importante para mi fue ampliar mi visión con todo lo que se puede hacer programando. <3
